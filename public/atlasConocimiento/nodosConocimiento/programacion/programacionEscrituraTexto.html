Una instrucción básica de programación es la que indica escritura de un texto.

Python
print("hola mundo");
Esta instrucción indica que se debe escribir el texto "Hola mundo".
Javascript
console.log("¡Hola mundo!");
Esta instrucción indica que se debe escribir el texto "Hola mundo".
Bash
echo "¡Hola mundo!"
Esta instrucción indica que se debe escribir el texto "Hola mundo".
