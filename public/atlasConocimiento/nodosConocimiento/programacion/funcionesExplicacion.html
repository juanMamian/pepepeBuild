En los programas informáticos frecuentemente se repiten series de instrucciones.

Para facilitar la escritura y análisis de los programas informáticos, es útil empaquetar intrucciones.

Un paquete de instrucciones se llama una función. Se le asigna un nombre a una función para referirse a ella en algún otro lugar del programa.

A veces se necesita usar datos específicos para ejecutar las instrucciones empaquetadas en una función. Para ello las funciones pueden crear variables en las cuales se puede almacenar y leer esos datos.

Las variables creadas para ser usadas durante la ejecución de una función se llaman parámetros de la función.
