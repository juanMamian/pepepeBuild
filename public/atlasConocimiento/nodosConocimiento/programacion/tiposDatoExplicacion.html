En un programa informático las variables pueden almacenar distintos tipos de datos.

Un número sin valores decimales, como el 1, 3, 5, etc., es un dato de tipo número entero. Frecuentemente se identifica como int Un número con valores decimales como el 1.5, 3.4, o 20.3432 es un dato de tipo número de punto flotante. Frecuentemente se identifica como float. Una letra o un símbolo, como a, r, %, (, etc. es un dato de tipo caracter. Frecuentemente se identifica como char Una palabra o texto como casa, carro, la casa, etc., es un dato de tipo Cadena de caracteres Frecuentemente se identifica como string. Un valor de verdad, como true o false es un tipo de dato Booleano. Frecuentemente se identifica como Bool o Boolean.

En algunos lenguajes de programación, al crear una variable se debe especificar el tipo de dato que se almacenará en ella.

int num;
Esta instrucción crea una variable llamada num que almacenara datos de tipo int.
float fracc;
Esta instrucción crea una variable llamada fracc que almacenara datos de tipo float.
string palabras;
Esta instrucción crea una variable llamada palabras que almacenara datos de tipo string.
char letra;
Esta instrucción crea una variable llamada letra que almacenara datos de tipo char.
bool presente;
Esta instrucción crea una variable llamada presente que almacenara datos de tipo boolean.
