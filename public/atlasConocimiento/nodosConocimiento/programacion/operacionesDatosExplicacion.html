Los procesadores pueden realizar operaciones entre datos.

Entre datos numéricos se pueden realizar operaciones aritméticas. Entre datos que son caracteres(char) o cadenas de caracteres (string), se pueden realizar concatenaciones. Entre datos booleanos se pueden realizar operaciones lógicas. Entre datos de cualquier tipo se pueden realizar comparaciones de igualdad y desigualdad. Entre datos numéricos se pueden realizar comparaciones mayor que y menor que.

Una instrucción de realizar una operación entre dos datos se llama una expresión.

Al ejecutar una expresión, el procesador obtiene un dato resultante. Esto se conoce como resolver la expresión.

15 + 10
Esta instrucción indica que se debe realizar una suma entre dos números.
Resolver
25
"Hogar " + "dulce hogar"
Esta instrucción indica que se debe concatenar el string hogar y el string dulce hogar
Resolver
"hogar dulce hogar"
9 / 2
Esta instrucción indica que se deben dividir los números 9 y 2.
Resolver
4.5
true && false
Esta instrucción indica que se deben realizar la operación lógica and entre los valores true y false.
Resolver
false
12 == "hola"
Esta instrucción indica que se deben realizar una comparación de igualdad entre el número 12 y la palabra hola.
Resolver
false
14 < 40
Esta instrucción indica que se debe realizar una de comparación menor qué entre el 14 y el 40.
Resolver
true
