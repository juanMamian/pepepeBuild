Los procesadores pueden ejecutar instrucciones escritas en lenguaje de máquina. Este lenguaje es difícil de leer para los seres humanos.

Para facilitar la escritura de programas informáticos se crearon lenguajes de programación, que son más fáciles de leer para los seres humanos.

Javascript
Es un lenguaje creado para hacer páginas web dinámicas y que con el tiempo ha sido usado para otras tareas relacionadas con la web.
C++
Es un lenguaje de uso general que fue creado para ampliar la utilidad del lenguaje C.
Python
Es un lenguaje de programación moderno famoso por ser amigable para principiantes.
Bash
Es un lenguaje de programación usado para hacer instrucciones dirigidas a sistemas UNIX.

Los procesadores no pueden leer las instrucciones escritas en los lenguajes de programación. Éstas deben ser traducidas al lenguaje de máquina antes de ser ejecutadas.

Algunos programas informáticos son completamente traducidos al lenguaje de máquina para generar un nuevo programa que sí puede ser enviado al procesador. Esto se llama compilar el programa. El software encargado de esta tarea se llama compilador.

Algunos programas informáticos son traducidos instrucción por instrucción: cada vez que se traduce una instrucción al lenguaje de máquina, es enviada al procesador y éste la ejecuta inmediatamente. Esto se llama interpretación del programa. El software encargado de esta tarea se llama intérprete.
