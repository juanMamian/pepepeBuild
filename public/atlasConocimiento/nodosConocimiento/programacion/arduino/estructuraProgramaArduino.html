Un programa arduino tiene la siguiente estructura fundamental. Presiona en cada zona para leer una explicación.
Cabecera
En esta zona se incluyen las librerias y se crean las variables u objetos que se usarán durante el programa.
Setup
Las instrucciones de esta zona son ejecutadas una vez al inicio del programa. Se usa para inicializar valores o realizar configuraciones.
Loop
Las instrucciones de esta zona son ejecutadas una y otra vez por el sistema. Aquí se escriben las instrucciones de operación.
#include "Stepper.h"

Stepper motor(12, 3, 4, 2, 1);
int direccion;

void setup(){
     Serial.begin(9600);
     Serial.println("Iniciando");
}

void loop(){
     motor.apuntar(20);
     delay(1000);
     motor.apuntar(120);
     delay(1000);
}
