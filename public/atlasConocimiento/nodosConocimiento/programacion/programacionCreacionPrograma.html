Para crear un programa informático se debe escribir un documento con instrucciones dirigidas al procesador usando algún lenguaje de programación.

Para ejecutar el programa escrito se puede escoger una de las siguientes opciones.

  • Ejecutar el programa usando un intérprete.
  • Compilar el programa para generar un nuevo documento escrito en lenguaje de máquina y, a continuación, ejecutarlo usando la interfaz ofrecida por el sistema operativo.
