Para computar información, los procesadores almacenan datos en su memoria para leerlos después y realizar acciones con ellos.

En los programas informáticos, se usa el concepto de variable para escribir instrucciones relacionadas con guardar y leer datos de la memoria.

Una variable es un lugar en la memoria dentro del cual se puede almacenar algún dato. Cada variable tiene un nombre. El procesador puede crear variables, asignar un dato para ser almacenado dentro de una variable o leer el dato almacenado dentro de una variable.
