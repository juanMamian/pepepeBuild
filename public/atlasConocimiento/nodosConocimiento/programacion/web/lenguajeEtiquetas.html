Un texto puede ser enriquecido si se introducen marcas que ayudan al lector a interpretarlo. El lenguaje de etiquetas es un sistema para marcar texto introduciendo etiquetas . Las etiquetas se escriben entre los símbolos: <> Puedes presionar sobre el texto para escuchar una interpretación.
<cantando> <gritando> <susurrando> Maestra Vida </cantando> </gritando> </susurrando>

Desmarcar
 Las etiquetas para marcar texto se escriben en pares.
  • La primera etiqueta indica el inicio de la marca.
  • La segunda etiqueta indica el final de la marca. Incluye el símbolo "slash" /.
Se puede aplicar varias marcas sobre el mismo texto.
<gritando><cantando> Maestra Vida </cantando></gritando>
Este texto está marcado para ser gritado y cantado.
