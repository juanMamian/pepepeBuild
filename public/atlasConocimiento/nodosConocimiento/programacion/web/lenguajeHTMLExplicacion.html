El lenguaje de marcado de hipertexto (HTML) es un lenguaje de etiquetas diseñado para ser interpretado por navegadores web.
  • center Texto centrado.
  • i Texto en itálica.
  • b Texto en negrilla.
  • h1 Cabecera de primer nivel.
  • h2 Cabecera de segundo nivel.
  • h3 Cabecera de tercer nivel.
  • p Párrafo.
  • mark Texto marcado.
  • s Texto tachado.
  • small Texto pequeño.
  • strong Texto importante.
  • sub Texto en sublinea.
Un navegador web puede leer un documento de texto e interpretar marcas HTML si las hay. Si se asigna una extensión html a un documento de texto, el sistema operativo de un computador sabrá que puede ser interpretado por un navegador.
