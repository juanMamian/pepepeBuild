En javascript se pueden usar funciones, que son paquetes de varias instrucciones en serie.

Las funciones son:
  • Declaradas: Se escoge un nombre para la función y se construye una serie de instrucciones que son contenidas por ella.
  • Invocadas o ejecutadas: Se ordena la ejecución de la función; es decir la ejecución de sus instrucciones internas.

Declaración de funciones

Una función se declara usando la palabra clave function seguida del nombre que se asignará a la función.

function oscurecer()
Se ha declarado una función de nombre "oscurecer".

Después se crea un espacio para escribir las instrucciones contenidas por esta función. Este espacio se llama el cuerpo de la función.

function oscurecer() {

}
Ahora las llaves indican el cuerpo de la función. En él se puede escribir instrucciones.
function oscurecer() {
document.querySelector("#boton1").style.backgroundColor = "white";
document.querySelector("#fondo").style.backgroundColor="black";
}
Ahora se han creado dos instrucciones dentro del cuerpo de la función.

Ejecución de funciones

Las funciones se ejecutan escribiendo el nombre de la variable que la contiene y los parámetros entre paréntesis. Si no hay parámetros, se deja vacío el espacio entre paréntesis

oscurecer();
Se ha invocado la función oscurecer; es decir que se ejecutarán las instrucciones declaradas en el cuerpo de la función.
