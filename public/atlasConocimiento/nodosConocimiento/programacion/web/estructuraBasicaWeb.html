Las páginas web se construyen sobre una estructura básica de elementos.
head
Este elemento contiene elementos de configuración de la página web.
<head>
<title>
</title>
title
El texto contenido dentro de este elemento será mostrado por el navegador como nombre de la página.
</head>
body
Cuerpo de la página. El contenido de este elemento será mostrado en el visor del navegador web.
<body>
</body>
