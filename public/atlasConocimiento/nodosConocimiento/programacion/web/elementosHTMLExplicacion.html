Las etiquetas HTML también pueden marcar zonas del documento web.

La etiqueta HTML más usada para marcar una zona del documento es la etiqueta div. La zona marcada no recibe inicialmente ningún trato especial. Las zonas del documento marcadas con una etiqueta HTML apropiada son llamadas elementos. Los elementos pueden anidarse unos dentro de otros. Los elementos de este documento están señalados por cuadros de colores. Dentro de ellos está el código que los genera (Pares de etiquetas div ) Puedes activarlos y desactivarlos.
