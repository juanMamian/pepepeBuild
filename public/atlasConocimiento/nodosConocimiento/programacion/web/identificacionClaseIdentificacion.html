Frecuentemente es útil hacer referencia a un elemento en particular. Para eso, los elementos pueden tener una palabra que los identifica. Como un nombre.

La palabra que identifica a un elemento se llama identificación del elemento.
Se abrevia id.

Para asignar identificacion a un elemento se escribe la palabra clave id dentro de la etiqueta inicial del elemento.

{{textoEtiquetaInicial1}}{{textoId}}{{textoEtiquetaInicial2}}
{{textoEtiquetaFinal}}
En la etiqueta inicial está ahora el atributo id.

La palabra de identificacion se asigna entre comillas usando el operador de asignación =.

{{textoEtiquetaInicial1+textoId}}{{textoAsign1}}menu{{textoAsign2}}{{textoEtiquetaInicial2}}
{{textoEtiquetaFinal}}
Ahora la palabra menu es la identificación del elemento.
