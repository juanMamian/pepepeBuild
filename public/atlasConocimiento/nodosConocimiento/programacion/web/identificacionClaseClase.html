Frecuentemente es útil hacer referencia a un grupo de elementos. Para eso, los elementos pueden pertenecer a una clase.

Para asignar una clase a un elemento se escribe la palabra clave class dentro de la etiqueta inicial del elemento.

{{textoEtiquetaInicial1}}{{textoClass}}{{textoEtiquetaInicial2}}
{{textoEtiquetaFinal}}
En la etiqueta inicial está ahora el atributo class.

El nombre de clase se asigna entre comillas usando el operador de asignación =.

{{textoEtiquetaInicial1+textoClass}}{{textoAsign1}}botones{{textoAsign2}}{{textoEtiquetaInicial2}}
{{textoEtiquetaFinal}}
Ahora el elemento es de clase botones.
Se pueden asignar varias clases en un elemento separadas con espacios.
{{textoEtiquetaInicial1+textoClass}}{{textoAsign1}} botones administracion {{textoAsign2}} {{textoEtiquetaInicial2}}
{{textoEtiquetaFinal}}
Este elemento es de clase botones y también de clase administracion
