Los elementos de una página web registran las acciones que ejecuta el usuario sobre ellos.

Algunos de los eventos que pueden realizarse sobre un elemento web son:

  • click: Un usuario hace click sobre el elemento.
  • dblclick: Un usuario hace doble click sobre el elemento.
  • focus: Un usuario selecciona el elemento.
  • blur: Un usuario deselecciona el elemento.
  • mouseenter: El cursor entra en el elemento.
  • mouseleave: El cursor sale del elemento.

Los eventos pueden ser manejados por funciones. Es decir: funciones que se ejecutan en respuesta a un evento.

Una función que responde a un evento se llama manejador del evento.

Registro de manejadores.

Para registrar un manejador para un evento, se empieza por escribir el nombre del evento precedido por la palabra on en la etiqueta inicial de un elemento.

{{textoEtiqueta}} onclick >
{{textoFinEtiqueta}}
Se ha creado el atributo onclick para referenciar que una función manejará el evento click en este elemento.

A continuación se asigna la función que manejará el evento.

{{textoEtiqueta}} onclick ="oscurecer()" >
{{textoFinEtiqueta}}
Se ha declarado que la función oscurecer manejará el evento click en este elemento.

Hecho el registro, ahora cada vez que ocurra el evento click en el elmento, la función oscurecer será ejecutada.
