Una instrucción fundamental consiste en cambiar la apariencia de un elemento. Para cambiar la apariencia de un elemento se cambia el valor de sus propiedades de estilo.

Una instruccion javascript que cambia la apariencia de un elemento, inicia con una referencia a dicho elemento.

document.querySelector("#boton1")
Se ha referenciado a un elemento cuya id es boton1.
Una vez que se ha escrito una referencia a un elemento, se accede a su propiedad "style" y, dentro de ella se accede a alguna de las propiedades de estilo.
document.querySelector("#boton1").style.width
Se ha accedido a la propiedad style del elemento y, dentro de ella, se ha accedido a la propiedad width del elemento referenciado.

Tras acceder a una propiedad de estilo, se puede escribir el operador de asignación = y, enseguida, el nuevo valor para ser asignado.

document.querySelector("#boton1").style.width = "400px"
Se ha asignado el valor 400px a la propiedad width en la propiedad style del elemento con id "boton1".
