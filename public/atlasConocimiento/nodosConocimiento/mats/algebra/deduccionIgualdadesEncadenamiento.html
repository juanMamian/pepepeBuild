Cuando a partir de una comparación se deduce una nueva, ésta es tan verdadera como la original.

Y si se toma la segunda comparación para deducir una tercera, dicha tercera comparación es tan verdadera como la segunda

Así que la tercera comparación es tan verdadera como la primera.

Deducir comparaciones y usarlas para hacer nuevas deducciones es una deducción encadenada.

\( 16 \times 2 = 32 \)
Deducir una nueva comparación
\( 16 \times 2 + 4 = 32 + 4 \)
Deducir una nueva comparación
\( {{{16 \times 2} + 4 } \over 2} = {{32 + 4} \over 2} \)
Deducir una nueva comparación
\( { { { {16 \times 2} + 4} \over 2 }- 10 } = {{{32 + 4} \over 2} }- 10 \)
