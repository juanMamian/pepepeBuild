Una variable es un símbolo usado para significar el concepto cualquier elemento de un conjunto. Frecuentemente se usa una letra, como \(x, y, w, v\) , etc. Frecuentemente la letra usada como variable corresponde a la letra usada para nombrar un conjunto. Así:
  • \( x \) significa Cualquier elemento del conjunto X
  • \( m \) significa Cualquier elemento del conjunto M
  • \( p \) significa Cualquier elemento del conjunto P
Si \(n\) es una variable que representa cualquier elemento del conjunto N, entonces:
  • \(n\) tiene más de tres letras significa cualquier elemento de N tiene más de tres letras.
  • \(n\) es menor que 10 significa cualquier elemento de N es menor que 10.
  • \(n\) es un número significa cualquier elemento de N es un número.

Si una afirmación es cierta para Cualquier elemento del conjunto, entonces también es cierta, si se escoge un elemento del conjunto.
